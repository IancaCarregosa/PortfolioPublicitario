🎨 Portfólio Publicitário — Gabriel Rocha

Este projeto é um site de portfólio desenvolvido para apresentar os trabalhos de um profissional da área de publicidade, design e social media. A aplicação foi construída com foco em estética, organização visual e experiência do usuário, simulando um portfólio real utilizado no mercado. O projeto inclui conteúdos fictícios (mock projects), permitindo futura substituição por trabalhos reais.

🧠 Sobre o projeto

O portfólio foi desenvolvido com o objetivo de:
Apresentar projetos de forma visual e organizada
Simular cases reais de branding e design
Criar uma navegação fluida e responsiva
Aplicar conceitos de UI/UX no front-end

🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla)
Bootstrap 5

🎯 Funcionalidades
Alternância entre modo claro e escuro
Carrosséis responsivos (Bootstrap)
Layout adaptado para mobile
Seção de portfólio com múltiplas categorias:
Posters
Convites
Branding & Identidade Visual
Fotografia
Página dinâmica para projetos individuais
Integração com contato (Email e WhatsApp)
